- 出版社：オライリー・ジャパン

## 📝 学習の目的と内容
本フォルダには、ニューラルネットワークの基礎から畳み込みニューラルネットワーク（CNN）まで、外部ライブラリに頼らずにアルゴリズムをスクラッチで実装したプロセスを記録しています。
単なるフレームワークの習得に留まらず、内部の計算ロジックを数式レベルで咀嚼することを目的に取り組みました。

### 主要な学習項目
- **パーセプトロンと活性化関数**: Sigmoid, ReLU, Softmax等の数学的性質と実装
- **誤差逆伝播法（Backpropagation）**: 計算グラフの概念を用いた連鎖律（Chain Rule）の理解と、各レイヤ（Affine, ReLU等）のクラス設計
- **学習の最適化**: 勾配降下法（SGD）に加え、Momentum, Adam等の最適化手法のアルゴリズム理解
- **モデルの評価と改善**: Batch Normalization, Dropout等を用いた過学習抑制と精度の検証
- **CNNの構造理解**: 畳み込み層（Convolution）およびプーリング層（Pooling）の仕組みと実装
