# Data Science Learning Journey & Technical Notes

本リポジトリは、データサイエンティストとしての専門性を支えるための「基礎理論の習得」および「技術検証」の記録をまとめたものです。
実務において「客観的なデータに基づいた意思決定」を支えるためには、ツールの習得だけでなく、その背後にあるアルゴリズムの仕組みを深く理解することが不可欠であると考え、日々の学習に取り組んでいます 。

## 📂 リポジトリの構成

各フォルダ内には、技術書や講座を通じて学んだ実装ロジックや、理論の検証プロセスを格納しています。

 
**`zero-dl-vol1`**: 『ゼロから作るDeep Learning』の学習記録。NumPyを用いたニューラルネットワークのスクラッチ実装 。


 
**`pytorch-advanced`**: 『つくりながら学ぶ！PyTorchによる発展ディープラーニング』の学習記録。VGG16を用いた転移学習やファインチューニングの実装演習 。



## ⚠️ 本リポジトリの性質について（免責事項）

* **学習のプロセスの記録**: 本リポジトリのコードは、主に技術書の写経や、それに基づいた挙動確認のための技術検証を目的としています。
* **オリジナル作品との区別**: これらは私のオリジナル作品ではなく、あくまで技術習得を目的とした「学習のプロセス」の記録であることを明記いたします。
* **データセットについて**: 著作権保護の観点から、書籍付属のデータセット等は含まれておりません。
