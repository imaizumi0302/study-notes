## 📝 学習の目的と内容
本フォルダでは、PyTorchを用いた画像認識タスクの基礎的な実装演習を記録しています。学習済みモデルを活用した「転移学習（Transfer Learning）」の初期設定プロセスを理解することを目的に取り組みました。

### 主要な学習項目
- **学習済みモデルの活用**: VGG-16ネットワーク構造のロード
- **転移学習の準備**: 出力層（全結合層）の付け替えの実装
- **パラメータのフリーズ**: 既存の学習済み重みを固定（`requires_grad = False`）し、特定の層のみを学習対象とする設定手法の習得
- **画像の前処理**: ネットワーク入力に合わせたリサイズおよび標準化処理の定義
